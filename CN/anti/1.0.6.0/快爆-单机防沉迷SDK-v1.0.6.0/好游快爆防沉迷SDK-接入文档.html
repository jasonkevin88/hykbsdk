1. SDK接入

1.1 Android Studio集成方式（推荐）

  1. SDK下载完，将m3839_fcm.aar文件导入项目的libs目录下
  2. 在主工程的build.gradle文件下添加以下代码即可完成依赖，
repositories {
   flatDir {
      dirs 'libs'
   }
}

然后引用aar文件

implementation(name: 'm3839_fcm', ext: 'aar')

2. 代码接入

2.1 sdk初始化

描述：

SDK的初始化方法

接口方法:

UnionFcmSDK.initSDK(Activity activity, UnionFcmParam param, UnionFcmListener listener);

参数定义：

参数名称 描述
activity 游戏主界面的Activity
config UnionFcmParam 防沉迷的参数信息
listener UnionFcmListener 检测回调监听

示例代码：

    String gameId = gameIdEdit.getText().toString();
    UnionFcmParam param = new UnionFcmParam.Builder()
                .setContact("test@qq.com")
                .setGameId(gameId)
                .setOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT)
                .build();
    UnionFcmSDK.initSDK(this, param, new UnionFcmListener() {
            @Override
            public void onFcm(int code, String message) {
                if(code == 100) {
                    //表示登录成功，获取登录信息
                    UnionFcmUser user = UnionFcmSDK.getUser();
                    if(user != null) {
                        Toast.makeText(SampleActivity.this, user.toString(), Toast.LENGTH_SHORT).show();
                    }
                }else if(2005 == code) {
                    finish();
                }else{
                    Toast.makeText(SampleActivity.this, message, Toast.LENGTH_SHORT).show();
                }
            }
        });

类描述：

防沉迷参数信息类：com.m3839.union.fcm.UnionFcmParam.Builder

方法名称 描述
setGameId(String gameId) 设置游戏ID，由快爆开发者平台上申请的参数
setContact(String contact) 设置开发者联系方式，用于玩家找回密码使用，必须填写
setDefaultPassword(String defaultPassword) 设置默认的初始化密码，不设置默认是3839 必须是4位数字
setOrientation(int orientation) 设置游戏屏幕方向

检测监听类： com.m3839.union.fcm.UnionFcmListener

方法名称 描述
onFcm(int code, String msg) 检测成功后回调方法

常见回调的code描述：

code 描述
-1 服务端请求异常
100 表示登录成功，根据个code，去获取用户信息UnionFcmSDK.getUser()
1001 青少年模式已关闭
1002 青少年模式已开启
1003 未设置找回密码联系方式
1004 青少年模式，“我知道了”的回调
1101 游戏不存在，未进行审核，联系运营人员处理
1102 游戏开发者需要，退出游戏操作或者注销账号
2001 用户认证失败
2002 登录失败
2003 取消登录
2004 登录异常
2005 游戏开发者需要，退出游戏操作或者注销账号

获取当前模式

该API在初始化之后调用，true：表示青少年模式，false：表示联网防沉迷

boolean isTeenMode = UnionFcmSDK.isTeenMode()

获取用户信息：

UnionFcmUser user = UnionFcmSDK.getUser()

类描述：

用户数据实体类： com.m3839.union.fcm.UnionFcmUser

方法名称 描述
getUserId() 获取用户uid
getNick() 获取用户昵称
getType() 获取登录方式
getAccessToken() 这个字段用来登录授权的有效性校验，accessToken有效期两小时

日志开关:

UnionFcmSDK.setDebug(true);//true：open log output

释放资源:

UnionFcmSDK.releaseSDK();

2.2 在Android 11 上无法唤起快爆app问题

如果开发者的targetSDKVersion 是30或者更高的话，需要在项目的AndroidManifest.xml文件添加如下代码：

  <!--适配android 11无法唤起第三方apk-->
  <queries>
    <!-- 好游快爆 -->
    <package android:name="com.xmcy.hykb" />
  </queries>

3. 混淆规则

如果您需要使用proguard混淆代码，需确保不要混淆SDK的代码。
请在proguard.cfg文件(或其它混淆文件)尾部添加如下配置:

# 好游快爆防沉迷DK
# 因为这些子类都有可能被外部调用
-keep public class * extends android.app.Activity
-keep public class * extends android.content.BroadcastReceiver
-keep public class * extends android.preference.Preference
-keep public class * extends android.view.View

# 保留在Activity中的方法参数是view的方法，
# 这样以来我们在layout中写的onClick就不会被影响
-keepclassmembers class * extends android.app.Activity{
    public void *(android.view.View);
}

# 不能混淆监听
-keeppackagenames com.m3839.union.fcm
-keep class com.m3839.union.fcm.** { *; }
-keep class com.m3839.fcm.sdk.LaunchActivity {
  public *;
}
